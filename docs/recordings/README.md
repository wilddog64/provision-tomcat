# Provision Tomcat Recordings

This directory contains recordings of various workflows and demos for the `provision-tomcat` role.

## Building Baseline Box

This recording shows the process of building the baseline Windows box (`windows11-tomcat112`) using `make vagrant-build-baseline`. It includes the installation of Java, Tomcat, Splunk Universal Forwarder, and security configurations.
